## Rede física

- Ethernet, Wi-Fi;
- MAC address;
- Física (Hardware) e Enlace (Software);

## Internet

- IP: Internet Protocol (Identifica a máquina dentro da rede);
	- Gateway;
	- Máscara de rede;
- Conexão entre redes;
- Transferência de Pacotes;

## Transporte

- TCP (Mais Confiável);
- UDP (Menos esforço);
- Comunicação Host-A-Host;
- Confiabilidade, integridade.

## Aplicação

- HTTP, FTP, SMTP
- Comunicação Processo-A-Processo
- Número da porta
