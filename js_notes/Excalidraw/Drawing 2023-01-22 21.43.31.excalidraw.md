# Text Elements
Prototype chain ^qJ46i8jx

The third one will refer to the second one and so fourth.
If you refer to an attribute that doesn't exist inside the
third one, it will look into the previous objects to find it. If 
don't find, return undefined  ^mwGUj5Bs

Object.prototype ^TDjWfqKa

All objects have and ultimate ancestor called
Object.prototype, a function that has a prototype
method. ^aI0mIJjm

If an attribute is declared in both the son and the ancestor,
the compiler will look for the son's value, since int came first. 
This is called "Shadowing". ^UIFaSyfn
